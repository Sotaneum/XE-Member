# XE-Member
Xpress Engine Default Board Skin


 - 기본 게시판 스킨을 이용한 Member 스킨입니다.

 - 내부에 첨부된 사진의 정보를 바탕으로 기본으로 보여주는 형태입니다.

 - 해당 Member의 정보를 게시글의 본문을 바탕으로 보여줍니다.

 - 게시판이기 때문에 이동이 편리하고 관리도 편리합니다.
